Detailed Description

Class that represent a remote node that will export some data (as BlueSTSDKFeature) that the user can query or ask to be notify when they change.

Note
In case that the node require the a secure connection with the pin it can happen that the notification subscription are sent before the connection completed. It is not possible have callback/notification when the connection completed. For this reason we try to subscribe the feature every second until we didn't receive some data from it.
Author
STMicroelectronics - Central Labs.
