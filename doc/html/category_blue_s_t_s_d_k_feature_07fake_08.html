- (NSData*) generateFakeData

abstract method, you have to overwrite this method if you want enable the possibility to use a fake feature into the ios emulator.

this method have to generate some random/fake data that will be used passed to the BlueSTSDKFeature::update:data:dataOffset:
Note
it is an abstract method, you have to overwrite it!
Returns
data that will be passed to the updateFeature for trigger a fake feature update
