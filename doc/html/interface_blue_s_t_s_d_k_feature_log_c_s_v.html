Detailed Description

class that implement the BlueSTSDKFeatureLogDelegate and store the data in a csv file with the same name of the feature.

it can be used for logging multiple feature, each feature has its file.The files are keeped open for avoid multiple fopen/fclose.

Note
Call closeFiles when you stop the logging for avoid to keep open the files

a log row has the format:

  • NodeName: name of the node that send the data
  • RawData: exadecimal version of the raw data used for extract the feature data
  • a colum for each feature item
